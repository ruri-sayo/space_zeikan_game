# Audit H680k v0 成果報告書

## 1. 実装概要
企画書_実装詳細.md を踏まえつつ、要望に合わせてゲーム体験を「物理学 + 証言」中心へ再設計し、ホーム画面/難易度選択付きで実装した。

## 2. 実装した主要機能
- ホーム画面の追加（Easy / Hard のモード選択）
- 10件の案件JSONデータ読み込み
- 監査エンジン（logic / physics）
- 判定エンジン（CLEAR / SECONDARY_AUDIT / DETAIN）
- スコア算出（検出70点 + 判定30点）
- 3カラムの監査コンソールUI
- 可視化パネルを Orbit / Signal に絞り込み（Hull削除）
- Easy: チェック項目ガイダンス + ヒント付き表示
- Hard: ヒント非表示（抽象チェック項目表示）
- 日本語UIボタン対応
- ホームへ戻るボタン + 戻る前確認ダイアログ
- 案件進行と結果モーダル

## 3. 技術構成
- Framework: Astro + React
- State: Zustand
- Validation: zod
- Logic rule: json-logic-js

## 4. 成果物
- `src/` 配下の実装一式
- `src/data/cases/` に10案件
- `Document/` 配下のドキュメント

## 5. 既知制約
- 可視化は v0 のためプレースホルダ中心（本格グラフ描画は次フェーズ）
- 証言矛盾はルールベースの簡易推論
- サーバー保存・認証なし（静的運用前提）

## 6. 次フェーズ提案
- Orbit/Signalの可視化品質向上
- 案件難易度調整とQA
- Playwright E2E 追加
- ブログ埋め込み時のパフォーマンス最適化
